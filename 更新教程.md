# 游戏更新教程 - 如何快速更新而不重新部署

## 🎯 本次更新内容

- **问题**：出牌后两边显示等待不同的人出牌
- **原因**：玩家切换和状态发送顺序错误

## 🚀 快速更新方法（推荐）

### 方法一：使用 SCP 命令更新（最简单）

**如果你使用的是 Mac 或 Linux：**

```bash
# 1. 打开终端，进入修改后的文件所在目录
# 假设修改后的文件在 ~/Downloads/robbery-game-fix/ 目录

cd ~/Downloads/robbery-game-fix/

# 2. 只上传修改的文件（game.js 和 style.css）
scp game.js style.css root@你的服务器IP:/root/robbery-game/

# 3. 登录服务器重启服务
ssh root@你的服务器IP
pm2 restart robbery-game
```

**如果你使用的是 Windows：**

使用 PowerShell 或 Git Bash：

```powershell
# 1. 进入文件所在目录
cd C:\Users\你的用户名\Downloads\robbery-game-fix

# 2. 上传修改的文件
scp game.js style.css root@你的服务器IP:/root/robbery-game/

# 3. 登录服务器重启
ssh root@你的服务器IP
pm2 restart robbery-game
```

### 方法二：使用 FTP 工具更新（图形界面）

**使用 FileZilla：**

1. 打开 FileZilla
2. 连接服务器（主机填IP，用户名root，密码，端口22）
3. 在右侧（服务器）找到 `/root/robbery-game/` 目录
4. 在左侧（本地）找到修改后的 `game.js` 和 `style.css`
5. **右键点击文件 → 上传**（会提示覆盖，选择是）
6. 打开终端连接服务器，执行：`pm2 restart robbery-game`

**使用 WinSCP：**

1. 打开 WinSCP
2. 连接服务器
3. 找到服务器上的 `/root/robbery-game/` 目录
4. 将本地的 `game.js` 和 `style.css` 拖过去覆盖
5. 打开终端重启服务：`pm2 restart robbery-game`

## 📋 完整更新脚本（一键更新）

创建一个 `update.sh` 脚本文件：

```bash
#!/bin/bash

# 配置
SERVER_IP="你的服务器IP"
SERVER_USER="root"
LOCAL_DIR="/path/to/your/local/game/files"
REMOTE_DIR="/root/robbery-game"

# 颜色输出
GREEN='\033[0;32m'
RED='\033[0;31m'
NC='\033[0m' # No Color

echo -e "${GREEN}开始更新游戏...${NC}"

# 上传修改的文件
echo "上传 game.js..."
scp "${LOCAL_DIR}/game.js" "${SERVER_USER}@${SERVER_IP}:${REMOTE_DIR}/"

echo "上传 style.css..."
scp "${LOCAL_DIR}/style.css" "${SERVER_USER}@${SERVER_IP}:${REMOTE_DIR}/"

# 重启服务
echo "重启游戏服务..."
ssh "${SERVER_USER}@${SERVER_IP}" "pm2 restart robbery-game"

echo -e "${GREEN}更新完成！${NC}"
```

使用方法：

```bash
# 1. 给脚本执行权限
chmod +x update.sh

# 2. 运行脚本
./update.sh
```

## 🆘 常见问题

### Q1: 更新后游戏还是不正常？

**可能原因**：浏览器缓存了旧代码

**解决方法**：
- 强制刷新页面：`Ctrl + F5`（Windows）或 `Cmd + Shift + R`（Mac）
- 或者清除浏览器缓存后重新打开

### Q2: 上传文件时提示 Permission denied？

**解决方法**：
```bash
# 登录服务器修改权限
ssh root@你的服务器IP
chmod 755 /root/robbery-game
chmod 644 /root/robbery-game/*.js /root/robbery-game/*.css
```

### Q3: 如何只更新一个文件？

```bash
# 只更新 game.js
scp game.js root@你的服务器IP:/root/robbery-game/
pm2 restart robbery-game
```

### Q4: 更新后想回滚到旧版本？

```bash
# 登录服务器
ssh root@你的服务器IP
cd /root/robbery-game

# 如果有备份
mv game.js game.js.new
mv game.js.bak game.js

# 重启服务
pm2 restart robbery-game
```

## 📝 建议：定期备份

在更新前，建议先备份服务器上的文件：

```bash
# 登录服务器
ssh root@你的服务器IP
cd /root/robbery-game

# 创建备份目录
mkdir -p backup/$(date +%Y%m%d)

# 备份当前文件
cp game.js style.css server.js backup/$(date +%Y%m%d)/

# 现在可以安全地更新了
```
