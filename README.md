# 抢劫艺术 · 联机版

一款支持远程联机的桌游网页应用。

## 🎮 游戏简介

共四轮：素描、雕塑、绘画、文物。

每人始终 5 张手牌：0-5 数字牌、老板牌、守卫犬牌、贪心贼牌。

轮到你时打出 1 张牌，根据效果从中央或别人那里拿赃物。

四轮后：先比不在场证明数量，最少者淘汰；其余再比总分。

## 🚀 快速开始

### 本地运行

```bash
npm install
npm start
```

然后访问 `http://localhost:3001`

### 远程联机

1. 将项目部署到服务器（详见部署教程.md）
2. 访问服务器地址
3. 选择「远程联机」模式
4. 创建房间并分享房间号给朋友

## 📱 游戏模式

- **本地联机**：多人同屏游戏，轮流操作
- **远程联机**：多人在线游戏，实时同步

## 🃏 卡牌说明

| 卡牌 | 图标 | 效果 |
|------|------|------|
| 数字牌 | 0️⃣-5️⃣ | 优先拿中央同数字，没有则可抢别人 |
| 老板牌 | 👩‍💼 | 若中央有老板则拿走，否则可抢别人老板 |
| 守卫犬 | 🐶 | 获得守卫犬，被抢时可挡刀 |
| 贪心贼 | 🦊 | 从中央任选一个赃物 |
