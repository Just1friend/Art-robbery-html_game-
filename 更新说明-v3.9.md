# 抢劫艺术 v1.2 更新说明

## 🐛 修复的Bug

### 1. 贪心贼选择后不能把牌拿到手上 ✅
- **问题**：贪心贼选择赃物后，牌没有正确添加到玩家手中
- **修复**：修改了 `resolveGreedyCard` 函数，确保选择后立即更新数据并执行飞行动画

### 2. 飞行动画只显示自己的 ✅
- **问题**：只能看见自己拿赃物时的飞行效果，看不见别人的
- **修复**：
  - 添加了 `getPlayerPositionElement` 辅助函数
  - 修改了 `takeLootFromCenter` 函数，使用正确的玩家位置元素
  - 现在所有人的赃物移动都有飞行动画

### 3. 文字提示只显示自己的行动 ✅
- **问题**：下面的文字提示只会显示自己的行动，不显示别人的
- **修复**：
  - 添加了 `actionHistory` 状态记录行动历史
  - 修改了 `logAction` 函数，显示最近3条行动记录
  - 添加了 `action-item` CSS样式

### 4. 本轮结束提示不消失 ✅
- **问题**：本轮结束的提示在下一轮开始后还存在
- **修复**：在 `startRound` 函数中添加隐藏提示的逻辑

### 5. 守卫犬机制错误 ✅
- **问题**：有守卫犬时，应该是被抢的人选择给什么，而不是抢的人选择
- **修复**：
  - 修改了 `robFromPlayer` 函数
  - 修改了 `robBossFromPlayer` 函数
  - 现在弹窗显示在被抢者的界面上，由被抢者选择

## ⚖️ 数值平衡调整

## 🎮 测试建议

更新后请测试以下功能：

1. **贪心贼测试**
   - 打出贪心贼，选择任意赃物
   - 确认赃物正确添加到本轮赃物中
   - 确认有飞行动画

2. **飞行动画测试**
   - 观察其他玩家拿赃物时的动画
   - 确认能从中央飞到对应玩家位置

3. **行动记录测试**

   - 确认能看到所有人的行动

4. **守卫犬测试**
   - 有守卫犬的玩家被抢时

5. **白点计算测试**
   - 游戏结束后查看白点统计
   - 确认按照新的规则计算
