# 抢劫艺术 - 京东云部署教程

## 📋 项目简介

这是一个支持远程联机的桌游网页应用，使用 Node.js + WebSocket 实现实时通信。

## 🔧 修复内容

本次更新修复了以下问题：

1. **贪心贼逻辑修复**：现在可以正确抢夺中央区域的任何赃物（包括老板牌），选择后能正确获得赃物
2. **手牌间距优化**：增加了手牌之间的间距，同时保持部分重合，手机端可以清晰看到数字
3. **数字牌emoji**：将数字牌图标从 `🔢` 替换为对应的数字emoji（0️⃣-5️⃣）
4. **牌堆洗牌机制**：当牌堆空了时，会自动将弃牌堆洗牌放回，确保每人始终有5张牌
5. **远程联机完善**：添加了心跳检测、更完善的状态同步和错误处理
6. **手机端适配**：优化了移动端布局和交互体验

## 🚀 京东云部署步骤

### 方法一：使用京东云轻量应用服务器（推荐）

#### 1. 购买服务器

1. 访问 [京东云官网](https://www.jdcloud.com/)
2. 选择「轻量应用服务器」
3. 选择配置（建议最低配置即可）：
   - CPU: 1核
   - 内存: 2GB
   - 带宽: 3Mbps
   - 系统: Ubuntu 20.04/22.04 或 CentOS 7/8

#### 2. 连接服务器

购买完成后，在控制台获取服务器的公网IP和初始密码，然后连接：

```bash
# Windows 使用 PowerShell 或 Git Bash
ssh root@你的服务器IP

# Mac/Linux 使用终端
ssh root@你的服务器IP
```

#### 3. 安装 Node.js

```bash
# Ubuntu/Debian 系统
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# CentOS/RHEL 系统
curl -fsSL https://rpm.nodesource.com/setup_18.x | bash -
yum install -y nodejs

# 验证安装
node -v
npm -v
```

#### 4. 上传项目文件

**方式A：使用 SCP 命令**

在本地电脑打开终端，进入项目文件夹：

```bash
# 将本地文件上传到服务器（在项目目录执行）
scp index.html game.js style.css package.json server.js root@你的服务器IP:/root/robbery-game/
```

**方式B：使用 FTP 工具**

1. 下载并安装 FileZilla 或 WinSCP
2. 连接服务器（协议选 SFTP，端口 22）
3. 在服务器创建 `/root/robbery-game/` 目录
4. 上传所有项目文件

**方式C：使用 Git（如果代码在GitHub）**

```bash
# 在服务器上执行
mkdir -p /root/robbery-game
cd /root/robbery-game
git clone 你的GitHub仓库地址 .
```

#### 5. 安装依赖并启动

```bash
cd /root/robbery-game

# 安装依赖
npm install

# 启动服务器（前台运行，测试用）
npm start

# 看到 "Server running on port 3001" 表示启动成功
```

#### 6. 配置防火墙

在京东云控制台：

1. 进入「安全组」设置
2. 添加规则：
   - 协议: TCP
   - 端口: 3001（或你设置的其他端口）
   - 源IP: 0.0.0.0/0（允许所有IP访问）

#### 7. 使用 PM2 后台运行（重要）

```bash
# 安装 PM2
npm install -g pm2

# 使用 PM2 启动
pm2 start server.js --name "robbery-game"

# 设置开机自启
pm2 startup
pm2 save

# 查看运行状态
pm2 status

# 查看日志
pm2 logs robbery-game

# 重启
pm2 restart robbery-game

# 停止
pm2 stop robbery-game
```

### 方法二：使用京东云容器服务（进阶）

#### 1. 创建 Dockerfile

在项目根目录创建 `Dockerfile`：

```dockerfile
FROM node:18-alpine

WORKDIR /app

COPY package*.json ./
RUN npm install --production

COPY . .

EXPOSE 3001

CMD ["node", "server.js"]
```

#### 2. 构建并推送镜像

```bash
# 登录京东云容器镜像服务
docker login 你的镜像仓库地址

# 构建镜像
docker build -t robbery-game:latest .

# 打标签
docker tag robbery-game:latest 你的镜像仓库地址/robbery-game:latest

# 推送
docker push 你的镜像仓库地址/robbery-game:latest
```

#### 3. 在京东云容器服务中部署

1. 创建 Kubernetes 集群
2. 创建工作负载，选择你的镜像
3. 配置服务暴露（LoadBalancer 或 NodePort）

## 🌐 访问游戏

部署完成后，访问地址：

```
http://你的服务器IP:3001
```

如果想使用域名，可以：

1. 购买域名并解析到服务器IP
2. 配置 Nginx 反向代理

### Nginx 配置示例

```bash
# 安装 Nginx
sudo apt-get install nginx  # Ubuntu
sudo yum install nginx      # CentOS
```

创建配置文件 `/etc/nginx/sites-available/robbery-game`：

```nginx
server {
    listen 80;
    server_name 你的域名;

    location / {
        proxy_pass http://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：

```bash
sudo ln -s /etc/nginx/sites-available/robbery-game /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

## 🛠️ 故障排查

### 1. 无法连接到服务器

```bash
# 检查服务器是否运行
curl http://localhost:3001

# 检查端口监听
netstat -tlnp | grep 3001

# 查看防火墙状态
sudo ufw status  # Ubuntu
sudo firewall-cmd --list-all  # CentOS
```

### 2. WebSocket 连接失败

- 确保服务器防火墙已开放对应端口
- 检查京东云安全组设置
- 查看服务器日志：`pm2 logs robbery-game`

### 3. 端口被占用

```bash
# 查找占用端口的进程
sudo lsof -i :3001

# 杀死进程
sudo kill -9 进程ID
```

### 4. 修改端口

编辑 `server.js` 文件：

```javascript
const PORT = process.env.PORT || 3001;  // 修改这里的数字
```

然后重启服务。
